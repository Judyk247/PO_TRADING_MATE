# 🚀 PO TRADING MATE

## Pocket Option Auto-Trading Bot

### Features
- ✅ Email/Password login to Pocket Option
- ✅ 4-Tab Web Interface (Platform, Robot, Support, Account)
- ✅ Custom Trading Strategy:
  - **1m/2m/3m**: Trend Following
  - **5m**: Reversal Trading
- ✅ Alligator Indicator + Fractals + Stochastic + EMA-150 + 3-Candle Patterns
- ✅ Martingale Recovery (2.3x multiplier, unlimited steps)
- ✅ Signals delivered 30 seconds before candle closes
- ✅ Auto-Trading with START/STOP buttons
- ✅ Manual CALL/PUT execution
- ✅ Filter assets by 85%+ payout
- ✅ Sound alerts for signals and trade results
- ✅ Trade statistics and live console log

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/po-trading-mate.git
cd po-trading-mate

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
